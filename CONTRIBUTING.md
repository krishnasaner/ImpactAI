# 🤝 Contributing to ImpactAI

Thank you for your interest in ImpactAI, a project maintained by Krishna Saner! We welcome pull requests, bug reports, and suggestions.

## How to Contribute

1. **Fork the Repository**
   Create a fork of the `ImpactAI` repository on your GitHub account.

2. **Clone the Repository**
   ```bash
   git clone https://github.com/krishnasaner/ImpactAI.git
   ```

3. **Create a Branch**
   Create a working branch for your feature or bug fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Changes**
   - Write clear, concise code.
   - We use React and TypeScript for the frontend, and Python (FastAPI) for the backend.
   - Remember to run linters and ensure the app boots successfully.

5. **Commit Your Code**
   Please write descriptive commit messages that explain *why* the change was made:
   ```bash
   git commit -m "feat: added new AI dashboard widget"
   ```

6. **Submit a Pull Request**
   Push your changes to your fork and submit a PR to the main branch. Let us know exactly what the PR resolves in the description.

## Code Standards
- Keep components modular.
- Remove redundant logs and commented-out code before pushing.
- Respect our inclusive, mental-health-aware environment.

Thank you for contributing to ImpactAI! — Krishna Saner
