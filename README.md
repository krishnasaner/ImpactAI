# ImpactAI

Final year project by Krishna Saner.

## Overview
ImpactAI is a platform designed to provide a robust, interactive dashboard and backend API for analytical problem-solving and insights tracking. It features automated AI-driven chat endpoints, seamless modern React dashboarding, predictive modeling capabilities, and mobile-friendly layouts.
